copiar los archivos a la carpeta $HOME (~) de Linux

al archivo createssh.sh y a updaterepo.sh dar permisos de ejecución:
cd ~
chmod +x createssh.sh
chmod +x updaterepo.sh


luego ejecutar ./createssh.sh

usar el nombre de archivo que se sugiere en el script. (algo como: /home/<usuario>/ssh/github_pcbog)

abrir https://github.com/settings/ssh/new y pegar el código ssh que arroja

reiniciar wsl

Para inicializar el repositorio por el motivo que sea
desde powershell, escribir

wsl git init
wsl git add README.md
wsl git commit -m "Initialized repository"
wsl git branch -m main
wsl git remote add origin git@github.com:pcbogota/mante.git
.\upload.ps1

ghconn realiza la conexión con github
updatelinux actualiza Linux fácilmente (requiere contraseña de sudo)
